# 🎓 Student Registration System using Java, JDBC & MySQL

<p align="center">
  <b>A menu-driven Student Registration System developed using Java, JDBC, and MySQL.</b><br>
  Perform complete CRUD (Create, Read, Update, Delete) operations with a simple console-based interface.
</p>

---

## 🚀 Project Overview

The **Student Registration System** is a Java console application that manages student records using **JDBC** and **MySQL**. It demonstrates complete **CRUD (Create, Read, Update, Delete)** operations with a menu-driven interface while storing records in a MySQL database.

This project was developed to strengthen concepts of:

- Java Programming
- JDBC Connectivity
- MySQL Database
- SQL CRUD Operations
- Object-Oriented Programming (OOP)

---

# ✨ Features

- ➕ Add Student
- 📋 View All Students
- 🔍 Search Student by ID
- ✏️ Update Student Details
- ❌ Delete Student
- 🗄️ MySQL Database Connectivity using JDBC
- 💻 Menu-Driven Console Application

# ▶️ How to Run

### Step 1

Clone the repository

```bash
git clone https://github.com/prince1830/Syntecxhub_Student_Registration_System.git
```

### Step 2

Open the project using **VS Code** or any Java IDE.

### Step 3

Create a MySQL database named:

```text
student_registration
```

### Step 4

Create the **students** table.

### Step 5

Update the database credentials in:

```text
DBConnection.java
```

### Step 6

Add the **MySQL Connector/J** library to the project.

### Step 7

Run:

```text
Main.java
```
